Help Center

Getting Started

Welcome to VHabits! Here's how to get started:

  • Create an account or log in
  • Add your first habit using the "Add New Habit" button
  • Track your daily progress
  • View your analytics and calendar

Managing Habits

You can:

  • Add new habits with custom colors and icons
  • Set weekly goals for each habit
  • Track completion daily
  • Edit or delete habits as needed

Analytics

Track your progress with:

  • Daily completion statistics
  • Weekly progress charts
  • Habit performance metrics

Need More Help?

VHabits is designed to be simple and intuitive. If you need additional assistance, please check our GitHub repository for documentation and updates.
